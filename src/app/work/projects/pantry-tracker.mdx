## Overview

Pantry Tracker is a responsive web application designed to help users manage their household pantry inventory and reduce food waste. Built with a modern full-stack architecture, it provides real-time collaboration features for shared households.

## Key Features

- **Real-time synchronization** using Firestore, enabling multiple users to share and update pantry lists simultaneously
- **RESTful APIs** for inventory management, expiration date tracking, and shopping list generation
- **Mobile-responsive UI** built with Material-UI components for seamless experience across devices

## Tech Stack

- **Frontend**: React, Material-UI
- **Backend**: Node.js, RESTful APIs
- **Database**: Firebase Firestore
- **Deployment**: Google Cloud Platform
