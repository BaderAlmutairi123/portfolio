## Overview

A comprehensive Linux system administration project focused on installing, configuring, and securing Linux environments for web hosting and enterprise use.

## Key Accomplishments

- **Web Server Setup**: Installed and configured Apache server with SSL certificates to ensure secure web hosting
- **Directory Services**: Set up and managed LDAP server for centralized authentication across the system
- **Network File System**: Configured NFS to enable efficient file sharing across networked systems
- **Performance Optimization**: Conducted system call tracing to analyze and optimize performance, ensuring secure and efficient operations

## Technologies

- Linux (Ubuntu/CentOS)
- Apache HTTP Server
- SSL/TLS Certificates
- OpenLDAP
- NFS (Network File System)
- strace / System Call Tracing
